This repository includes files related to the manuscript: "Quantification of glass fibers from environmental samples", Kozloski et al.

Included here are example FTIR spectra from pennate diatoms collected in the Mekong River, Cambodia and in Lake Tahoe, Nevada, USA, as well 
as example spectra collected from aging fiber glass products and pristine glass fibers intended for use in Fiberglass repairs. 

The SOP document (provided in HTML and PDF format) provides R code for analysis of raw FTIR spectra to differentiate glass fibers from diatom frustules. 
However, as described in the manuscript, this analysis assumes that particles have been previously classified as "mineral" using a reliable FTIR matching
library, and screened for complicance with the morphology criteria (translucent, uncurved and untapered, cylindrical fibers).
